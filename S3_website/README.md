# S3 WebSite

Crear un sitio web usando S3

## Variables
### Obligatorias
```
region = "string"
// Nombre del bucket
name = "string"

```
### Opcionales
```
s3_acl = "string | default = public-read"
// Ruta de la pagina de inicio
index = "string | default = index.html"
// Ruta de la pagina de error 404
error = "string | default = index.html"
```
